## О проекте

`SyntheticHealthSimulator` позволяет генерировать реалистичные (с точки зрения статистических закономерностей) панельные
данные о состоянии здоровья виртуальной популяции на протяжении 20 лет.

Каждый «пациент» имеет демографические характеристики, генетический профиль, историю образа жизни (питание, физическая
активность, курение, сон, стресс) и
ежегодно измеряемые биомаркеры (вес, давление, холестерин, HbA1c).

На основе этих данных рассчитываются риски развития различных заболеваний (ССЗ, диабет 2 типа, инсульт, НАЖБ,
колоректальный рак, цирроз) и моделируются исходы.

Основное назначение — создание качественных датасетов для:

- обучения моделей прогнозирования рисков;
- исследования влияния факторов образа жизни;
- тестирования алгоритмов обработки пропущенных данных (MAR);
- экспериментов с балансом классов.

## Использование

Базовый запуск генерации данных (5000 человек, 20 лет наблюдения) выполняется командой:

```bash
python generator.py
```

Все настройки задаются в функции `main()`:

- `N_PEOPLE` — количество синтетических пациентов.
- `YEARS` — период наблюдения (лет).
- `seed` — случайное зерно для воспроизводимости.
- `diabetes_threshold` — порог калибровки риска диабета (по умолчанию 18.7, обеспечивает prevalence ~15%).

После завершения работы в папке `data/synthetic` будут созданы пять CSV-файлов:

1. `01_cohort_baseline.csv` — начальные характеристики когорты.
2. `02_lifestyle_history.csv` — ежегодная история образа жизни.
3. `03_biomarkers_history.csv` — ежегодные значения биомаркеров с шумом.
4. `04_health_risks.csv` — рассчитанные риски и бинарные исходы заболеваний.
5. `05_aggregated_dataset_with_missing.csv` — агрегированный датасет (одна строка на пациента) с синтезированными
   пропусками (MAR).

Для каждого файла также генерируется JSON-метаданные.

## Структура генерируемых данных

### Базовая когорта (`01_cohort_baseline.csv`)

- `person_id`, `age_start`, `sex`, `height_cm`, `weight_start_kg`, `bmi_start`, `muscle_mass_start`, `bmr_start_kcal`,
  `base_calories`
- Генетические риски по 6 заболеваниям (`genetic_risk_*`)
- Начальные значения биомаркеров (`initial_*`)

### История образа жизни (`02_lifestyle_history.csv`)

Для каждого года (0…19) сохраняются:

- `age_{year}`
- `weight_kg_true_{year}`, `muscle_mass_factor_true_{year}`, `bmr_kcal_true_{year}`, `tdee_kcal_true_{year}`
- Параметры питания: `protein_pct_{year}`, `fat_pct_{year}`, `carb_pct_{year}`, `saturated_fat_pct_{year}`,
  `simple_carbs_pct_{year}`, `fiber_g_day_{year}`, `sodium_mg_day_{year}`, `kbju_imbalance_{year}`,
  `simple_carbs_g_{year}`
- Образ жизни: `alcohol_g_per_week_{year}`, `cardio_met_minutes_{year}`, `strength_met_minutes_{year}`,
  `total_met_minutes_{year}`, `cigarettes_per_day_{year}`, `stress_level_{year}`, `sleep_hours_{year}`,
  `cumulative_smoking_{year}`

### Биомаркеры (`03_biomarkers_history.csv`)

Для каждого года: `weight_kg_{year}`, `hdl_mgdl_{year}`, `total_cholesterol_mgdl_{year}`, `sbp_mmhg_{year}`,
`hba1c_percent_{year}`, `bmi_{year}`, `bmi_corrected_{year}`, `non_hdl_mgdl_{year}`, `non_hdl_mmol_{year}`,
`muscle_mass_factor_{year}`

### Риски и исходы (`04_health_risks.csv`)

- Риски (`*_risk_10year`) и бинарные флаги (`has_*`) для CVD, диабета, инсульта, НАЖБ, колоректального рака, цирроза.
- `health_score`, `primary_death_cause`, `estimated_event_age`, `sex`

### Агрегированный датасет (`05_aggregated_dataset_with_missing.csv`)

- Усреднённые и стартовые/конечные значения факторов образа жизни (`avg_*`, `*_start`, `*_end`, `*_change`).
- Финальные биомаркеры (`final_*`).
- Риски и исходы.
- Синтезированные пропуски (MAR) в колонках `final_sbp_mmhg`, `final_hdl_mgdl`, `final_total_cholesterol_mgdl`,
  `final_hba1c_percent`, `avg_alcohol_g_per_week`, `avg_total_met_minutes`.

## Калибровка и настройка

Все коэффициенты в модели намеренно снижены по сравнению с эпидемиологическими данными, чтобы предотвратить
доминирование одного фактора и обеспечить сбалансированное обучение ML-моделей. Ключевые калибровочные параметры:

- **Порог диабета** (`diabetes_threshold`) — регулирует распространённость диабета. Значение 18.7 даёт ~15%
  заболеваемости за 20 лет.
- **Интерцепт SCORE2** (`ALPHA_SCORE2 = -3.8`) — задаёт базовый уровень сердечно-сосудистого риска.
- **Коэффициенты курения, алкоголя, натрия** — уменьшены для баланса признаков.

Более подробное описание всех формул, коэффициентов и допущений приведено в [научной справке](SCIENTIFIC_REFERENCE.md) (
файл в репозитории).

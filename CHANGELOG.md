## [1.0.0] - 2026-02-27

### Added
- Initial release of SyntheticHealthSimulator
- Synthetic medical data generation for 6 diseases (CVD, diabetes, stroke, NAFLD, colorectal cancer, cirrhosis)
- 20-year panel data with lifestyle, biomarkers, and genetic risk factors
- MAR missing value simulation (2-15%)
- Kaggle-ready CSV export with JSON metadata
- Scientific reference documentation

### Technical Details
- Python 3.14+
- Dependencies: numpy, pandas

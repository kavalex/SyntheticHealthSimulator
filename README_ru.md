## О проекте

`SyntheticHealthSimulator` генерирует реалистичные (с точки зрения статистических закономерностей) панельные данные о
состоянии здоровья виртуальной популяции на протяжении 20 лет. Каждый «пациент» имеет:

- **Демографические характеристики** (возраст, пол, рост);
- **Генетический профиль** (полигенные риски);
- **Историю образа жизни** (питание, физическая активность, курение, сон, стресс);
- **Ежегодно измеряемые биомаркеры** (вес, артериальное давление, холестерин, HbA1c).

На основе этих данных рассчитываются риски развития различных заболеваний (сердечно-сосудистые заболевания, диабет 2
типа, инсульт, НАЖБП, колоректальный рак, цирроз) и моделируются исходы.

**Основная цель** — создание качественных наборов данных для обучения моделей прогнозирования, исследования влияния
факторов образа жизни и тестирования алгоритмов обработки пропущенных данных.

## Обзор пайплайна

1. **Генератор** (`generator.py`)
    - Базовая когорта (демография, генетика, исходные биомаркеры)
    - 20-летние траектории образа жизни (процессы ОУ)
    - Ежегодная эволюция биомаркеров с шумом
    - Расчет рисков заболеваний (SCORE2, FINDRISC-подобные)

2. **Выходные наборы данных** (5 CSV + 5 JSON)
    - `01_cohort_baseline` — неизменяемые характеристики
    - `02_lifestyle_history` — ежегодные факторы образа жизни
    - `03_biomarkers_history` — ежегодные биомаркеры с шумом
    - `04_health_risks` — 10-летние риски и исходы
    - `05_aggregated_dataset_with_missing` — готово для ML с пропусками MAR

3. **Валидация** (`validator.py`)
    - Структура и полнота файлов
    - Физиологические диапазоны (жёсткое ограничение)
    - Временные тренды и корреляции
    - Распространённость заболеваний и коморбидность
    - Паттерны пропусков MAR и согласованность между наборами

4. **Анализ** (5 Jupyter-ноутбуков)
    - `01`: Анализ базовой когорты
    - `02`: Анализ траекторий образа жизни
    - `03`: Анализ трендов и шума биомаркеров
    - `04`: Валидация и прогнозирование моделей рисков
    - `05`: Полный ML-пайплайн (импутация, инженерия, ансамбль)

5. **ML-модели готовы к обучению**
    - Классификация (6 заболеваний)
    - Регрессия (прогнозирование биомаркеров)
    - Тестирование импутации пропущенных данных
    - Анализ важности признаков

## Использование

### Базовая генерация данных

Генерация данных для 5000 человек с периодом наблюдения 20 лет:

```bash
python generator.py
```

Все настройки определены в функции `main()`:

| Параметр               | Описание                                 | Значение по умолчанию  |
|------------------------|------------------------------------------|------------------------|
| `target_n_people`      | Количество синтетических пациентов       | 5000                   |
| `years`                | Период наблюдения (лет)                  | 20                     |
| `seed`                 | Случайное зерно для воспроизводимости    | 42                     |
| `diabetes_threshold`   | Порог калибровки риска диабета           | 16.5 (~15% prevalence) |
| `boundary_coefficient` | Коэффициент оверсемплинга для фильтрации | 2.41                   |

После завершения в папке `data/synthetic_v*` будут созданы **пять CSV-файлов** (и соответствующие JSON-файлы
метаданных).

### Валидация данных

Репозиторий включает встроенный валидатор (`validator.py`) для обеспечения соответствия сгенерированных данных ожидаемым
статистическим свойствам, физиологическим диапазонам и корреляциям.

**Что проверяет валидатор:**

- Полнота и структура файлов
- Демографические распределения (возраст, пол, рост)
- Распределения генетических рисков
- Диапазоны биомаркеров и жёсткое ограничение (только финальный год)
- Временные тренды (например, снижение HDL, рост HbA1c)
- Распространённость заболеваний и корреляции риск-исход
- Коморбидность (2 и более заболевания)
- Паттерны пропусков MAR (Missing-at-Random)
- Согласованность между наборами данных (person IDs, BMI)

**Запуск валидации:**

```bash
python validator.py --local data/synthetic_v*
```

Опциональный флаг `--report report.json` сохраняет подробный JSON-отчёт:

```bash
python validator.py --local data/synthetic_v* --report validation_report.json
```

Если все проверки пройдены, вы увидите:

```
ALL CHECKS PASSED SUCCESSFULLY!
```

### Анализ чувствительности

Инструмент анализа чувствительности (`sensitivity_analysis.py`) включён для тестирования стабильности генератора на
разных случайных зёрнах и оценки согласованности производительности ML-моделей.

**Запуск анализа чувствительности:**

```bash
python sensitivity_analysis.py
```

Это выполнит:

- Генерацию данных с несколькими случайными зёрнами
- Обучение классификатора Gradient Boosting на каждом наборе данных
- Расчёт AUC-оценок и распространённости заболеваний
- Генерацию сводной статистики и визуализаций
- Сохранение результатов в `sensitivity_analysis_results.csv` и `sensitivity_analysis_plots.png`

## Ноутбуки для анализа

Репозиторий включает пять комплексных Jupyter-ноутбуков для исследования данных, валидации и демонстрации ML-пайплайна.
Каждый ноутбук фокусируется на определённом аспекте сгенерированных данных.

### Ноутбук 1: Анализ базовой когорты (`01_cohort_baseline_analysis.ipynb`)

**Цель:** Исследовать статические характеристики пациентов и оценить, насколько хорошо базовые данные предсказывают
заболевания, развитые за 20 лет.

**Ключевые анализы:**

- Демографические распределения (возраст, пол, рост)
- Распределения генетических рисков
- Диапазоны исходных биомаркеров и соответствие спецификациям генератора
- Корреляционный анализ между базовыми признаками
- Прогнозирование заболеваний только по базовым данным (Логистическая регрессия, Random Forest)
- Анализ важности признаков

**Ключевые выводы:**

- Базовые признаки сами по себе дают низкую предсказательную способность (AUC 0.51-0.62)
- Подтверждает, что исходы в основном определяются образом жизни, накопленным за 20 лет
- Возраст, BMI и исходный HbA1c — наиболее важные базовые предикторы

**Выходные файлы:** `output/01_cohort_baseline/`

---

### Ноутбук 2: Анализ истории образа жизни (`02_lifestyle_history_analysis.ipynb`)

**Цель:** Анализировать 20-летние траектории образа жизни, проверить параметры процессов ОУ и связать траектории с
демографией и исходами.

**Ключевые анализы:**

- Верификация процессов ОУ (автокорреляция, оценки theta)
- Обнаружение стрессовых событий и анализ длительности
- Кластеризация траекторий (KMeans по среднему, std, тренду)
- Различия образа жизни по полу и статусу заболевания
- Прогнозирование будущего образа жизни из прошлых траекторий
- Добавленная стоимость статических признаков vs. временные ряды

**Ключевые выводы:**

- Аэробная и силовая активность показывают реалистичное стохастическое поведение
- Стрессовые события сохраняются в следующий год (эффект затухания подтверждён)
- Три различных кластера образа жизни с разной распространённостью заболеваний
- Траектории образа жизни — основные драйверы будущих исходов (R2 approx 0.50-0.60)

**Выходные файлы:** `output/02_lifestyle_history/`

---

### Ноутбук 3: Анализ биомаркеров (`03_biomarkers_analysis.ipynb`)

**Цель:** Исследовать ежегодные измерения биомаркеров с добавленным шумом, проверить жёсткое ограничение и
проанализировать временные тренды.

**Ключевые анализы:**

- Верификация жёсткого ограничения по всем 20 годам
- Оценка измерительного шума (с использованием истинных значений для веса)
- Временные тренды по полу и статусу диабета
- Удаление шума с помощью гауссовского сглаживания
- Прогнозирование будущих значений биомаркеров из истории
- Межбиомаркерные корреляции во времени

**Ключевые выводы:**

- Соответствие года 19: 100% в физиологических диапазонах
- Наблюдаемый шум соответствует ожидаемым значениям (вес sigma approx 1.5 кг)
- HDL снижается approx 10 мг/дл за 20 лет
- SBP увеличивается approx 11 мм рт.ст. за 20 лет
- HbA1c увеличивается approx 2.8% за 20 лет
- Шум ослабляет корреляции (истинные связи сильнее наблюдаемых)

**Выходные файлы:** `output/03_biomarkers/`

### Ноутбук 4: Анализ рисков для здоровья (`04_health_risks_analysis.ipynb`)

**Цель:** Валидировать формулы рисков, проанализировать калибровку и построить модели прогнозирования для всех 6
заболеваний.

**Ключевые анализы:**

- Валидация диапазонов рисков (жёсткое ограничение)
- Согласование риск-распространённость
- Метрики калибровки (оценка Брайера, отношение E/O, наклон калибровки)
- Распределения рисков по демографии (пол, квинтили возраста)
- Корреляционный анализ (риски vs. признаки)
- ML-прогнозирование для всех 6 заболеваний (Логистическая регрессия, Random Forest, Gradient Boosting)
- Многоклассовое прогнозирование (любое заболевание)
- Проверка согласованности между наборами данных

**Ключевые выводы:**

- Все 6 диапазонов рисков проходят верификацию жёсткого ограничения
- Согласование риск-распространённость: максимальная разница менее 0.05
- Оценки Брайера от 0.126 (Диабет) до 0.211 (Рак)
- Отношения E/O близки к 1.0: диапазон 0.991-1.022
- Лучшая модель ССЗ: Логистическая регрессия AUC-ROC = 0.651
- Многоклассовая классификация: Macro F1 = 0.16, Accuracy = 0.27

**Выходные файлы:** `output/04_health_risks/`

### Ноутбук 5: Полный ML-пайплайн (`05_aggregated_complete_pipeline.ipynb`)

**Цель:** Продемонстрировать полный цикл ML на агрегированном наборе данных с пропусками MAR.

**Ключевые анализы:**

- EDA: Анализ пропущенных значений (валидация паттерна MAR)
- Валидация спецификаций генератора
- Проверка согласованности между наборами данных
- Подготовка данных: целевая переменная и признаки (удаление утечек)
- Сравнение методов импутации (Простая vs. MICE)
- Инженерия признаков (признаки на основе предметной области)
- Выбор модели с кросс-валидацией
- Ансамбль: Voting Classifier
- Оценка и сравнение моделей
- Внешняя валидация (синтетические vs. референсные данные)
- Анализ порогов для клинической полезности

**Ключевые выводы:**

- 5000 пациентов, 47 признаков после очистки
- 6 колонок с пропусками MAR: 2.9-3.7%
- Распространённость диабета: approx 15%
- Импутация MICE рекомендована для данных MAR
- Добавлено 5 инженерных признаков (metabolic_syndrome_score, high_alcohol_risk и др.)
- Лучшая модель: Ансамбль (AUC-ROC approx 0.65)
- Улучшение vs. Ноутбук 4: +0.06 (+10%)
- Рекомендация порогов: 0.2 (скрининг), 0.3 (баланс), 0.4 (диагностика)

**Выходные файлы:** `output/05_aggregated_pipeline/`

## Структура сгенерированных данных

Все файлы, кроме агрегированного, имеют панельную структуру: одна строка или набор показателей хранится для каждого
пациента и года наблюдения.

### 1. Базовая когорта (`01_cohort_baseline_v*.csv`)

Одна строка на пациента. Содержит неизменяемые характеристики и исходные значения.

| Категория                  | Колонки                                                                                                                                                  |
|----------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------|
| Идентификатор и демография | `person_id`<br/>`age_start`<br/>`sex`<br/>`height_cm`                                                                                                    |
| Антропометрия              | `weight_start_kg`<br/>`bmi_start`<br/>`muscle_mass_start`                                                                                                |
| Энергетический метаболизм  | `bmr_start_kcal`<br/>`base_calories`                                                                                                                     |
| Генетические риски         | `genetic_risk_cvd`<br/>`genetic_risk_diabetes`<br/>`genetic_risk_stroke`<br/>`genetic_risk_nafld`<br/>`genetic_risk_cancer`<br/>`genetic_risk_cirrhosis` |
| Исходные биомаркеры        | `initial_hdl_mgdl`<br/>`initial_total_cholesterol_mgdl`<br/>`initial_sbp_mmhg`<br/>`initial_hba1c_percent`                                               |
| Латентные факторы          | `lipid_factor`                                                                                                                                           |

### 2. История образа жизни (`02_lifestyle_history_v*.csv`)

Показатели образа жизни и расчётные параметры записываются для каждого пациента и каждого года.

| Категория                        | Колонки                                                                                                                                                                                                                                         |
|----------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| Идентификатор, год, возраст      | `person_id`<br/>`age_{year}`                                                                                                                                                                                                                    |
| Физические показатели (истинные) | `weight_kg_true_{year}`<br/>`muscle_mass_factor_true_{year}`<br/>`bmr_kcal_true_{year}`<br/>`tdee_kcal_true_{year}`                                                                                                                             |

| Образ жизни                      | `alcohol_g_per_week_{year}`<br/>`cardio_met_minutes_{year}`<br/>`strength_met_minutes_{year}`<br/>`total_met_minutes_{year}`<br/>`cigarettes_per_day_{year}`<br/>`stress_level_{year}`<br/>`sleep_hours_{year}`<br/>`cumulative_smoking_{year}` |

### 3. Биомаркеры (`03_biomarkers_history_v*.csv`)

Ежегодные значения биомаркеров (с добавленным шумом).

| Категория          | Колонки                                                                                                                       |
|--------------------|-------------------------------------------------------------------------------------------------------------------------------|
| Идентификатор, год | `person_id`<br/>`age_{year}`                                                                                                  |
| Биомаркеры         | `weight_kg_{year}`<br/>`hdl_mgdl_{year}`<br/>`total_cholesterol_mgdl_{year}`<br/>`sbp_mmhg_{year}`<br/>`hba1c_percent_{year}` |
| Производные        | `bmi_{year}`<br/>`bmi_corrected_{year}`<br/>`non_hdl_mgdl_{year}`<br/>`non_hdl_mmol_{year}`<br/>`muscle_mass_factor_{year}`   |

### 4. Риски и исходы (`04_health_risks_v*.csv`)

Рассчитанные 10-летние риски и бинарные флаги наличия заболеваний за весь период наблюдения.

| Категория     | Колонки                                                                                                                                                       |
|---------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------|
| Идентификатор | `person_id`                                                                                                                                                   |
| Риски         | `cvd_risk_10year`<br/>`diabetes_risk_10year`<br/>`stroke_risk_10year`<br/>`nafld_risk_10year`<br/>`colorectal_cancer_risk_10year`<br/>`cirrhosis_risk_10year` |
| Исходы        | `has_cvd`<br/>`has_diabetes`<br/>`has_stroke`<br/>`has_nafld`<br/>`has_colorectal_cancer`<br/>`has_cirrhosis`                                                 |
| Дополнительно | `health_score`<br/>`primary_death_cause`<br/>`estimated_event_age`<br/>`sex`                                                                                  |

### 5. Агрегированный набор данных с пропусками (`05_aggregated_dataset_with_missing_v*.csv`)

Одна строка на пациента. Содержит усреднённые и начальные/конечные значения факторов образа жизни, финальные биомаркеры,
риски, исходы и синтезированные пропущенные значения (через механизм MAR).

**Пропущенные значения могут появляться в колонках:** `final_sbp_mmhg`, `final_hdl_mgdl`,
`final_total_cholesterol_mgdl`, `final_hba1c_percent`, `avg_alcohol_g_per_week`, `avg_total_met_minutes`.

| Категория                        | Колонки                                                                                                                                                                                                                        |
|----------------------------------|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| Идентификатор и демография       | `person_id`<br/>`sex`<br/>`age_start`<br/>`age_end`                                                                                                                                                                            |
| Усреднённые факторы образа жизни | `avg_alcohol_g_per_week`<br/>`avg_total_met_minutes`<br/>`avg_sleep_hours`<br/>`avg_stress_level`<br/>`avg_fiber_g_day`<br/>`avg_sodium_mg_day`<br/>`avg_saturated_fat_pct`<br/>`avg_simple_carbs_g`<br/>`avg_macro_imbalance` |
| Динамика веса и мышечной массы   | `bmi_start`<br/>`final_bmi`<br/>`weight_start_kg`<br/>`final_muscle_mass`                                                                                                                                                      |

## Калибровка и конфигурация

Все коэффициенты в модели **намеренно снижены** по сравнению с эпидемиологическими данными для предотвращения
доминирования одного фактора и обеспечения сбалансированного обучения ML-моделей.

### Ключевые настраиваемые параметры:

| Параметр               | Описание                                 | Значение по умолчанию     |
|------------------------|------------------------------------------|---------------------------|
| `diabetes_threshold`   | Регулирует распространённость диабета    | 16.5 (~15%)               |
| `cvd_intercept`        | Свободный член модели SCORE2             | -3.8 (~15-18% prevalence) |
| `stroke_intercept`     | Свободный член модели инсульта           | -3.0 (~18-20% prevalence) |
| `boundary_coefficient` | Коэффициент оверсемплинга для фильтрации | 2.41                      |

**Целевая распространённость заболеваний:**

- ССЗ: ~15-18%
- Диабет: ~15%
- Инсульт: ~18-20%
- Коморбидность (2 и более заболевания): 15-50%

Остальные коэффициенты (SCORE2, курение, алкоголь, натрий) подробно описаны
в [Научном описании](SCIENTIFIC_REFERENCE_RU.md).

# Научная справка по модели генерации синтетических медицинских данных

## Введение

Представленная модель генерации синтетических медицинских данных основана на механистическом подходе, объединяющем
физиологические модели метаболизма, эмпирические эпидемиологические данные и клинические шкалы оценки рисков. Модель
предназначена для имитации влияния образа жизни на здоровье человека в течение 20 лет. Все коэффициенты основаны на
мета-анализах, систематических обзорах и калиброваны для обеспечения реалистичности ежегодных изменений.

**Проект:** `SyntheticHealthSimulator`

**ВАЖНО**: Данная модель содержит научно обоснованные упрощения и теоретические допущения, сделанные специально для
создания учебного синтетического датасета для обучения ML-моделей. Она НЕ предназначена для клинического применения и
замены профессиональной медицинской оценки.

## 1. Физиологические основы модели

### 1.1 Расчет базового метаболизма (BMR)

Модель использует формулу Mifflin-St Jeor:

**Формула Mifflin-St Jeor:**

- Мужчины: $BMR_{\text{базовый}} = 10 \times \text{вес (кг)} + 6.25 \times \text{рост (см)} - 5 \times \text{возраст (лет)} + 5$
- Женщины: $BMR_{\text{базовый}} = 10 \times \text{вес (кг)} + 6.25 \times \text{рост (см)} - 5 \times \text{возраст (лет)} - 161$

**Коррекция на мышечную массу:**

$BMR_{\text{скорректированный}} = BMR_{\text{базовый}} + 8.5 \times (\text{MuscleMass}_{\text{кг}} - \text{TypicalMuscleMass}_{\text{кг}})$

где $\text{TypicalMuscleMass} = 0.3 \times \text{вес тела}$ (типичная мышечная масса составляет ~30% веса), а 8.5
ккал/кг/день — разница в энергозатратах между мышечной и жировой тканью в покое.

***Примечание:** Значение 0.3 × вес тела используется как теоретический эталон для расчёта отклонения; фактическая
мышечная масса в данных может быть существенно ниже (см. раздел 1.4). Это упрощение принято для учебных целей и не
влияет на относительные сравнения.*

*Примечание:* Формула Mifflin-St Jeor уже включает возрастную коррекцию. Дополнительное снижение метаболизма с возрастом
моделируется через изменение мышечной массы (раздел 1.4).

### 1.2 Расчет суточной потребности в энергии (TDEE)

Для учета физической активности используется система MET-минут:

$TDEE = BMR_{\text{скорректированный}} \times \text{фактор активности}$

**Факторы активности по калиброванной классификации:**

| MET-минут/неделю | Фактор | Описание                              |
|------------------|--------|---------------------------------------|
| &lt; 450         | 1.2    | Сидячий/легкая активность (1-2 дня)   |
| 450-900          | 1.375  | Умеренная активность (3-5 дней)       |
| 900-1500         | 1.55   | Высокая активность (6-7 дней)         |
| &gt; 1500        | 1.725  | Очень высокая активность (спортсмены) |

*Калибровка:* Пороги скорректированы с исходной классификации ВОЗ по дням недели на MET-минуты для обеспечения
реалистичного распределения населения (~40% сидячие, ~35% легкая, ~20% умеренная, ~5% высокая активность). Фактор 1.9
для экстремальных нагрузок (&gt;3000 MET-мин/нед) **не используется** — верхний порог ограничен 1.725 для предотвращения
нефизиологических TDEE &gt;4000 ккал.

### 1.3 Модель изменения массы тела

#### 1.3.1 Калорийный баланс

Изменение веса рассчитывается по механистической модели за временной шаг $\Delta t = 365$ дней (1 год):

$\Delta W = \frac{(C - TDEE) \times 365}{7700}$

где:

- $\Delta W$ — изменение веса (кг) за год
- $C$ — потребление калорий (ккал/день)
- $TDEE$ — расход калорий (ккал/день)
- $7700$ — калорийный эквивалент 1 кг жировой ткани

#### 1.3.2 Распределение изменения веса

$\text{ProteinFactor} = \min(0.4, \; 0.1 + 0.05 \times \text{StrengthActivity} + 0.05 \times \text{ProteinBalance})$

$\Delta W_{\text{мышцы}} = \Delta W \times \text{ProteinFactor}$

$\Delta W_{\text{жир}} = \Delta W \times (1 - \text{ProteinFactor})$

где $\text{StrengthActivity}$ — количество силовых тренировок в неделю (0-5, рассчитывается
как $\text{strength\_met} / 150$), а $\text{ProteinBalance}$: 1 при потреблении &gt;1.6 г/кг, 0 при 0.8-1.6, -1 при
&lt;0.8.

*Примечание:* Максимальная доля мышц в приросте веса ограничена 40% для предотвращения физиологически невозможных
сценариев. При дефиците калорий мышечная масса может уменьшаться.

#### 1.3.3 Упрощенная модель состава тела

**Важное допущение:** Модель использует упрощенное представление состава тела:

$\text{Вес} = \text{Жировая масса} + \text{Мышечная масса} + \text{Константа}$

где **Константа** (масса костей, органов, воды) считается неизменной на протяжении моделирования. Это упрощение
необходимо для вычислительной эффективности и фокуса на динамике жировой и мышечной массы как ключевых факторов
метаболизма и рисков. В реальности масса костей и органов также подвержена небольшим изменениям, но их вклад в
вариативность рисков минимален по сравнению с жировой и мышечной тканью.

Расчетная формула обновления веса:
$\text{Вес}_{t+1} = (\text{Жировая масса}_t + \Delta W_{\text{жир}}) + (\text{Мышечная масса}_t + \Delta W_{\text{мышцы}}) + \text{Константа}$

### 1.4 Модель мышечной массы

**Мышечная масса в кг** рассчитывается как:

$\text{MuscleMass}_{t+1} = \text{MuscleMass}_t + \Delta W_{\text{мышцы}}$

где $\Delta W_{\text{мышцы}}$ — изменение из раздела 1.3.2.

**Индекс мышечной массы** (MuscleMassFactor, 0-20) используется для коррекции ИМТ:

$\text{MuscleMassFactor} = \text{MuscleMass}_{\text{кг}} / 0.5$ (1 единица индекса = 0.5 кг)

$\text{ИМТ}_{\text{скорр}} = \text{ИМТ} \times (1 - 0.01 \times \text{MuscleMassFactor})$

***Примечание:** В начальной когорте индекс мышечной массы генерируется как:*

- *для мужчин: $\mathcal{N}(12, 3^2)$, ограничение [0,20];*
- *для женщин: $\mathcal{N}(8, 2.5^2)$, ограничение [0,20].*
  *Таким образом, средняя мышечная масса в кг составляет 6 кг у мужчин и 4 кг у женщин, что соответствует упрощённой
  модели состава тела.*

## 2. Модели биомаркеров

### 2.1 Систолическое артериальное давление (САД)

#### 2.1.1 Формула изменения САД (Годовое изменение)

$\Delta SBP = \beta_{\text{ИМТ}} \times \Delta BMI + \beta_{\text{Na}} \times \Delta Na_{100mg} + \beta_{\text{Alc}} \times \text{AlcEffect} + \beta_{\text{Stress}} \times \Delta Stress + \beta_{\text{Sleep}} \times \text{SleepEffect}$

#### 2.1.2 Модель влияния алкоголя (Калиброванная линейная)

**Хроническое влияние (среднесуточное потребление):**
$$
\text{AlcEffect} =
\begin{cases}
0, & \text{если } Alc_{g/day} \leq 10 \\
0.03 \times (Alc_{g/day} - 10), & \text{если } Alc_{g/day} &gt; 10
\end{cases}
$$

*Калибровка:* Коэффициент 0.03 мм рт.ст. на 1 г/день сверх порога 10 г является сниженным для баланса признаков в
ML-модели. Эпидемиологические данные (Xin et al., 2001) указывают на эффект ~0.1 мм рт.ст./г, но в модели используется
сниженное значение для предотвращения доминирования фактора алкоголя.

#### 2.1.3 Влияние сна (теоретическое допущение)

$\text{SleepEffect} = \max(0, 6 - \text{AvgSleepHours}) \times 0.5$

#### 2.1.4 Таблица коэффициентов влияния на САД:

| Фактор   | Коэффициент | Единица измерения                  | Научный источник          | Примечание                       |
|----------|-------------|------------------------------------|---------------------------|----------------------------------|
| ИМТ      | +0.7        | мм рт.ст. на 1 кг/м²               | INTERSALT Study           | —                                |
| Натрий   | **+0.25**   | мм рт.ст. на 100 мг/день           | He et al., 2013 (Adj.)    | *Калибровано: оригинал ~0.5-1.0* |
| Алкоголь | **+0.03**   | мм рт.ст. на 1 г/день сверх порога | Xin et al., 2001 (Adj.)   | *Калибровано: оригинал ~0.1*     |
| Стресс   | +0.5        | мм рт.ст. на 1 единицу шкалы       | Everson-Rose et al., 2004 | —                                |
| Сон      | +0.5        | мм рт.ст. на час дефицита сна      | Теоретическое допущение   | —                                |

### 2.2 Холестерин ЛПВП

#### 2.2.1 Формула изменения ЛПВП (Годовое изменение)

$\Delta HDL = +0.2 \times \left(\frac{CardioActivity}{1000}\right) + 0.3 \times \left(\frac{StrengthActivity}{450}\right) - 0.05 \times \Delta SFA_{ppt} + 0.1 \times Fiber_{1g} + 0.05 \times \text{MuscleMassFactor}$

где $StrengthActivity$ измеряется в MET-минутах, а 450 MET-минут = 3 тренировки по 150 MET-минут при умеренной
интенсивности. **$SFA$ (saturated fat) передается как доля (0.0–1.0), $\Delta SFA_{ppt}$ — изменение в процентных
пунктах ($\Delta SFA \times 100$).**

**Таблица коэффициентов влияния на ЛПВП:**

| Фактор                | Коэффициент | Единица измерения                      | Научный источник        |
|-----------------------|-------------|----------------------------------------|-------------------------|
| Кардио активность     | +0.2        | мг/дл на 1000 MET-минут                | Kodama et al., 2007     |
| Силовые тренировки    | +0.3        | мг/дл на 450 MET-минут (≈3 тренировки) | Теоретическое допущение |
| Насыщенные жиры (SFA) | -0.05       | мг/дл на 1 процентный пункт            | Mensink et al., 2003    |
| Клетчатка             | +0.1        | мг/дл на 1г                            | Brown et al., 1999      |
| Мышечная масса        | +0.05       | мг/дл на 1 ед. индекса                 | Теоретическое допущение |

*Примечание:* SFA хранится и передается в модели как доля (например, 0.30 для 30%). Для расчета изменения используется
перевод в процентные пункты: $\Delta SFA_{ppt} = \Delta SFA \times 100$.

### 2.3 Гликированный гемоглобин (HbA1c)

#### 2.3.1 Формула изменения HbA1c (Годовое изменение)

$\Delta HbA1c = 0.15 \times \Delta SimpleCarbs_{10g} + 0.03 \times \Delta BMI - 0.002 \times Fiber_{1g} - 0.01 \times Activity_{1000MET} + \mathbf{0.002} \times \text{KBJU\_Imbalance} - 0.005 \times \text{MuscleMassFactor} + 0.01 \times \text{SleepDeficit}$

*Калибровка:* Коэффициент для КБЖУ **0.002 применяется на 1 единицу индекса** (0-100 шкала), максимальный вклад при
KBJU=100 составляет +0.2%. Коэффициент снижен с 0.02 (первоначальная спецификация) для предотвращения доминирования
фактора (макс вклад 2% → 0.2%) и обеспечения баланса признаков.

**Ограничение диапазона:** Значения HbA1c жёстко ограничены диапазоном 3.5–12%. Значения &gt;12% соответствуют тяжёлому,
неконтролируемому диабету и маловероятны для 20-летней когорты без летального исхода. В коде:
`HbA1c = clip(calculated, 3.5, 12.0)`

#### 2.3.2 Индекс дисбаланса КБЖУ

Рекомендованный баланс: Белки 30%, Жиры 30%, Углеводы 40%.

**Расчет индекса:**

$\text{raw\_KBJU} = \sqrt{(\Delta P)^2 + (\Delta F)^2 + (\Delta C)^2}$

где:

- $\Delta P = |\text{Факт\_Белки\%} - 30|$
- $\Delta F = |\text{Факт\_Жиры\%} - 30|$
- $\Delta C = |\text{Факт\_Углеводы\%} - 40|$

**Нормализация:**

$\text{KBJU\_Imbalance} = \min(100, \text{raw\_KBJU} \times 10)$

Итоговый индекс в диапазоне 0-100 используется в расчетах. Умножение на 10 с ограничением 100 является техническим
решением для совместимости с логистической моделью риска, не клинической шкалой.

**Таблица коэффициентов влияния на HbA1c:**

| Фактор                | Коэффициент | Единица измерения          | Научный источник          |
|-----------------------|-------------|----------------------------|---------------------------|
| Простые углеводы      | +0.15       | % на 10г                   | Brand-Miller et al., 2009 |
| ИМТ                   | +0.03       | % на 1 кг/м²               | Abdullah et al., 2010     |
| Клетчатка             | -0.002      | % на 1г                    | Weickert et al., 2006     |
| Физическая активность | -0.01       | % на 1000 MET              | Boule et al., 2001        |
| Дисбаланс КБЖУ        | +0.002      | % на 1 ед. индекса (0-100) | Теоретическое допущение   |
| Мышечная масса        | -0.005      | % на 1 ед. индекса         | Теоретическое допущение   |
| Дефицит сна           | +0.01       | % при сне &lt;6 часов      | Теоретическое допущение   |

### 2.4 Общий холестерин и non-HDL

**Формула изменения Total cholesterol (Годовое изменение):**

$\Delta TC = +1.5 \times \Delta SFA_{ppt} - 0.5 \times Fiber_{1g} - 0.3 \times \left(\frac{Activity_{total}}{1000}\right)$

где $\Delta SFA_{ppt}$ — изменение в процентных пунктах (например, с 0.30 до 0.31 = +1 ppt). **SFA передается в модели
как доля (0.0–1.0), поэтому формула в коде:** $1.5 \times (\Delta SFA \times 100)$.

***Примечание:** Генетический риск влияет только на базовый уровень TC (при инициализации), не на годовое изменение.
В коде к базовому уровню TC добавляется $0.8 \times \text{GeneticRisk}_{\text{CVD}}$ (в мг/дл), после чего применяется
clipping 150–350 мг/дл. Это отражает биологическую предрасположенность, а не скорость изменения под воздействием
факторов риска.*

**Расчет non-HDL cholesterol:**

1. Рассчитать TC в мг/дл по приведенной формуле
2. Рассчитать HDL в мг/дл по формуле из раздела 2.2
3. **non-HDL (мг/дл) = TC - HDL**
4. **non-HDL (ммоль/л) = (TC - HDL) / 38.67** (конвертация)

*Примечание:* При сильном шуме измерений возможны отрицательные промежуточные значения; в коде применяется
`clip(non_hdl_mgdl, 50, 232)` перед конвертацией в ммоль/л. Полученное значение в ммоль/л затем дополнительно обрезается
до диапазона [2.0, 6.0] (см. раздел 8.3).

**Таблица коэффициентов влияния на Total cholesterol:**

| Фактор                | Коэффициент | Единица измерения           | Научный источник    | Примечание             |
|-----------------------|-------------|-----------------------------|---------------------|------------------------|
| Насыщенные жиры (SFA) | +1.5        | мг/дл на 1 процентный пункт | Clarke et al., 1997 | —                      |
| Клетчатка             | -0.5        | мг/дл на 1г                 | Brown et al., 1999  | —                      |
| Физическая активность | -0.3        | мг/дл на 1000 MET-минут     | Kodama et al., 2007 | —                      |
| Генетика              | +0.8        | мг/дл на 1 ед. генет.       | Hindy et al., 2020  | Только базовый уровень |

### 2.5 Шум измерений и биологическая вариативность

Для обеспечения реалистичности данных и предотвращения переобучения ML-моделей на идеальных формулах, к финальным
значениям биомаркеров добавляется случайный шум.

**Обоснование:** Лабораторные анализы имеют погрешность, а физиологические показатели fluctuate ежедневно (биологическая
вариативность).

**Формула шума:**

$Value_{final} = Value_{calculated} + \mathcal{N}(0, \sigma^2)$

**Параметры шума ($\sigma$):**

- САД: $\sigma = 5$ мм рт.ст.
- ЛПВП: $\sigma = 5$ мг/дл
- Total Cholesterol: $\sigma = 10$ мг/дл
- HbA1c: $\sigma = 0.2$ %
- Вес: $\sigma = 1.5$ кг

*Примечание:* Hard clipping применяется после добавления шума для обеспечения физиологических ограничений.

## 3. Модели расчета медицинских рисков

### 3.1 Сердечно-сосудистые заболевания (SCORE2-подобная модель)

**Упрощенная логистическая модель SCORE2:**

$CVD_{\text{logit}} = \alpha + \beta_1 \times (Age-40) + \beta_2 \times (SBP-120) + \beta_3 \times (\text{non-HDL}-3.5) + \beta_4 \times Smoking + \beta_5 \times Diabetes + \beta_6 \times GeneticRisk_{CVD}$

Где коэффициенты:

- $\alpha = \mathbf{-3.8}$ (Интерцепт калибровки)
- $\beta_1 = 0.05$ (возраст за год)
- $\beta_2 = 0.03$ (САД за мм рт.ст.)
- $\beta_3 = 0.3$ (non-HDL за ммоль/л)
- $\beta_4 = \mathbf{0.08}$ (курение за пачко-год, калибровано)
- $\beta_5 = 0.9$ (наличие диабета)
- $\beta_6 = 0.4$ (генетический риск ССЗ)

**Обоснование калибровки:**

- Интерцепт -3.8 обеспечивает ожидаемую заболеваемость ССЗ ~15-20% за 20 лет в популяции 35±8 лет.
- Коэффициент курения 0.08 снижен относительно эпидемиологических данных (0.6) для предотвращения доминирования фактора
  курения и обеспечения баланса признаков для ML. При стаже 20 пачко-лет вклад составит $1.6$ (OR $\approx 5.0$), что
  создаёт различимый, но не подавляющий эффект.

**Расчет риска:**

$Risk_{CVD} = \frac{1}{1 + e^{-CVD_{\text{logit}}}}$

**Примечание:** Значение non-HDL в ммоль/л перед подстановкой в формулу дополнительно обрезается до [2.0, 6.0] в
соответствии с разделом 8.3.

### 3.2 Диабет 2 типа (FINDRISC-подобная модель)

**Система баллов:**

| Фактор                                      | Баллы |
|---------------------------------------------|-------|
| Возраст &gt; 45 лет                         | 2     |
| ИМТ 25-29.9                                 | 1     |
| ИМТ ≥ 30                                    | +2    |
| HbA1c 5.7-6.4%                              | 3     |
| HbA1c ≥ 6.5%                                | +5    |
| Курение                                     | 2     |
| Низкое потребление клетчатки (&lt;20г/день) | 1     |
| Дисбаланс КБЖУ &gt; 20 ед.                  | 2     |
| Мышечная масса &lt; 5 ед.                   | 1     |
| Дефицит сна (&lt;6 часов)                   | 2     |
| Генетический риск диабета &gt; 1.5          | 3     |

**Формула расчета риска:**

$$Risk_{Diabetes} = \frac{1}{1 + e^{-0.4 \times (\text{TotalScore} - \text{threshold})}}$$

где `threshold` — калибровочный параметр, позволяющий регулировать общую распространённость диабета в синтетической
популяции. Значение по умолчанию (`threshold = 18.7`) обеспечивает prevalence диабета около 15% за 20-летний период,
что соответствует целевым показателям для ML-задач.

*Калибровка:* Коэффициент 0.4 и пороговое значение подобраны эмпирически для получения реалистичного распределения
рисков с учётом дополнительных факторов (КБЖУ, мышечная масса, сон), расширяющих оригинальную шкалу FINDRISC.

### 3.3 Дополнительные модели рисков (расширение)

Модель включает дополнительные упрощенные логистические модели для обогащения датасета многоклассовыми задачами:

**Инсульт:**

$$\text{logit} = 0.03 \times (Age-50) + 0.015 \times (SBP-120) + 0.3 \times (HbA1c-5.5) + 0.15 \times Smoking + 0.03 \times (BMI-25) + 0.4 \times Genetic - 3.0$$

**НАЖБ** (неалкогольная жировая болезнь печени):

$$\text{logit} = 0.04 \times (BMI-25) + 0.25 \times (HbA1c-5.5) + 0.4 \times (SFA-0.3) + 0.3 \times [\text{Fiber}&lt;25] + 0.4 \times Genetic - 2.5$$

**Колоректальный рак:**

$$\text{logit} = 0.04 \times (Age-50) + 0.6 \times [\text{Fiber}&lt;20] + 0.4 \times [Alc&gt;140] + 0.5 \times Smoking + 0.3 \times [BMI&gt;30] + 0.7 \times Genetic - 1.5$$

**Цирроз:**

$$\text{logit} = 0.4 \times \ln(Alcohol+1) + 0.2 \times Smoking + 0.3 \times [BMI&gt;30] + 0.6 \times Genetic - 2.0$$

*Примечания:*

- Эти модели являются **теоретическими допущениями** для обогащения датасета. Коэффициенты не калиброваны по
  эпидемиологическим данным в отличие от SCORE2 и FINDRISC.
- Отдельный генетический риск ожирения не реализован, так как ожирение моделируется через динамический ИМТ и факторы
  образа жизни.

### 3.4 Полигенные риски

#### 3.4.1 Генерация профиля SNP

Для каждого пациента генерируется набор из 15 SNP (5 для ССЗ, 5 для диабета, 5 для ожирения). Каждый SNP имеет:

- Частоту аллеля риска: случайная от 0.1 до 0.5
- Отношение шансов (OR): случайное от 1.1 до 1.3

#### 3.4.2 Расчет полигенного риска

Для каждого заболевания:

$\text{GeneticRisk} = \prod_{i=1}^{5} OR_i^{G_i}$

где $G_i$ — количество аллелей риска (0, 1, 2) для i-го SNP.

**Нормализация:**

$\text{GeneticRisk}_{\text{норм}} = \frac{\text{GeneticRisk}}{\text{СреднийRisk}}$

где $\text{СреднийRisk}$ — средний риск в популяции, рассчитанный при средних частотах аллелей.

#### 3.4.3 Распределение рисков

- ССЗ: $\text{GeneticRisk}_{\text{CVD}} \sim \text{Логнормальное}(0, 0.4^2)$, обрезанное [0.5, 2.0]
- Диабет: $\text{GeneticRisk}_{\text{Diabetes}} \sim \text{Логнормальное}(0, 0.3^2)$, обрезанное [0.5, 2.0]
- Ожирение: Не моделируется отдельно — учитывается через ИМТ, мышечную массу и генетические риски
  сопутствующих заболеваний (NAFLD, диабет 2 типа)

*Примечание:* В коде используется упрощенная логнормальная генерация без явного моделирования отдельных SNP. Это
соответствует целевому балансу между сложностью и вариативностью для ML-обучения.

## 4. Временные паттерны образа жизни

### 4.1 Автокоррелированные временные ряды (Процесс Орнштейна-Уленбека)

Для генерации реалистичных траекторий используется модель случайного блуждания с возвратом к среднему:

$X_{t+1} = X_t + \theta \times (\mu - X_t) + \sigma \times \epsilon_t$

где:

- $\theta$ — скорость возврата к среднему (0.1-0.3)
- $\mu$ — долгосрочное среднее
- $\sigma$ — волатильность
- $\epsilon_t \sim \mathcal{N}(0,1)$

**Конкретные параметры OU-процессов, используемые в коде:**

| Фактор   | $\theta$ | $\sigma$ | Min | Max  |
|----------|----------|----------|-----|------|
| Алкоголь | 0.2      | 0.2      | 0   | 500  |
| Кардио   | 0.2      | 0.2      | 0   | 3000 |
| Силовые  | 0.25     | 0.25     | 0   | 1500 |
| Курение  | 0.3      | 0.3      | 0   | 40   |
| Стресс   | 0.2      | 1.0      | 0   | 10   |
| Сон      | 0.2      | 0.8      | 4   | 10   |

### 4.2 Сезонные и случайные события

**Примечание:** Сезонные колебания **не реализованы в коде** (осознанное упрощение). При годовой дискретизации модели
сезонные вариации усредняются и не оказывают значимого влияния на долгосрочные траектории. OU-процессы (раздел 4.1)
обеспечивают достаточную вариативность без явного моделирования сезонности.

**Случайные стрессовые события:**

**Вероятность стрессового события в любой год составляет 0.4 (в среднем раз в 2.5 года).** События (развод, потеря
работы, болезнь) влияют на:

- Уровень стресса: +3 балла на **1 год с затуханием** (+1.5 на следующий год)
- Качество сна: -1 час на 1 год
- Потребление алкоголя: +20% на 1 год

*Упрощение:* Длительность увеличена с 6 месяцев (первоначальная спецификация) до 1 года с затуханием для соответствия
годовой дискретизации модели.

## 7. Методология генерации данных

### 7.1 Иерархическая модель

Генетика → Демография → Образ жизни → Биомаркеры → Риски → Исходы

### 7.2 Формулы эволюции параметров

**Временной шаг:** Все изменения биомаркеров и веса рассчитываются на период $\Delta t = 365$ дней (1 год).

**Эволюция веса и состава тела:**

$W_{t+1} = W_t + \Delta W_{\text{жир}} + \Delta W_{\text{мышцы}}$

где $\Delta W_{\text{жир}}$ и $\Delta W_{\text{мышцы}}$ из раздела 1.3.2.

**Эволюция мышечной массы:**

$\text{MuscleMass}_{t+1} = \text{MuscleMass}_t + \Delta W_{\text{мышцы}}$

### 7.3 Генерация начальных условий

Для каждого синтетического пациента:

1. Генерация демографических данных (возраст, пол)
2. Генерация генетического профиля (15 SNP, упрощенная модель)
3. Расчет начальных биомаркеров на основе демографии и генетики
4. Назначение начального образа жизни (случайное с учетом корреляций)

## 8. Ограничения модели

### 8.1 Упрощения

- **Линейность**: $\Delta Biomarker = \sum \beta_i \times Factor_i$ (упрощение для учебной модели)
- **Аддитивность**: Эффекты факторов суммируются (упрощение для ML-обучения)
- **Независимость**: Корреляции между факторами упрощены
- **Типы активности**: Разделение на кардио и силовые тренировки является теоретическим допущением
- **non-HDL**: В SCORE2 используется measured non-HDL, в модели он рассчитывается упрощенно как TC - HDL
- **Total cholesterol**: Упрощенный расчет без разделения на LDL и VLDL
- **SCORE2 модель**: Упрощенная логистическая модель аппроксимирует оригинальную полиномиальную
- **Состав тела**: Упрощенная модель с постоянной массой костей/органов (см. раздел 1.3.3)

### 8.2 Теоретические допущения (нужно учитывать при интерпретации)

1. **Влияние мышечной массы** на BMR, HDL и HbA1c — теоретические коэффициенты, не подтвержденные крупными
   мета-анализами
2. **Влияние сна** на САД и HbA1c — основано на ассоциативных исследованиях, механистическая связь упрощена
3. **Индекс дисбаланса КБЖУ** — оригинальная метрика модели, не валидирована в клинических исследованиях; нормализация (
   ×10, ограничение 100) является техническим решением
4. **Влияние силовых тренировок на HDL** — коэффициент +0.3 мг/дл является теоретическим допущением, калибровано через
   450 MET-минут
5. **Полигенная модель риска** — упрощенное представление реальной полигенной архитектуры заболеваний (логнормальное
   распределение вместо явного моделирования SNP)
6. **Отсутствие сезонности** — годовая дискретизация модели делает сезонные колебания несущественными; вариативность
   обеспечивается OU-процессами
7. **Дополнительные модели рисков** (инсульт, НАЖБ, рак, цирроз) — теоретические допущения для ML-обучения, не
   калиброваны по эпидемиологическим данным
8. **Упрощенный состав тела** — масса костей и органов считается константой (см. раздел 1.3.3)

### 8.3 Диапазоны валидности

- Возраст: 20-70 лет
- ИМТ: 16-50 кг/м²
- ЛПВП: 20-100 мг/дл
- Total cholesterol: 150-350 мг/дл
- non-HDL: **в мг/дл: 50–232; в ммоль/л: 2.0–6.0**. После расчёта non-HDL (мг/дл) = TC - HDL применяется
  clipping [50, 232]. Затем полученное значение делится на 38.67 и ещё раз обрезается до [2.0, 6.0] ммоль/л (это может
  привести к несоответствию с нижней границей в мг/дл, что является осознанным упрощением).
- САД: 80-200 мм рт.ст.
- **HbA1c: 3.5-12%** (hard clip, значения &gt;12% исключены как несовместимые с жизнью)
- Дисбаланс КБЖУ: 0-100 ед. (нормализованная шкала)
- Индекс мышечной массы: **0–20, средние значения: мужчины 12, женщины 8 (см. раздел 1.4).**

### 8.4 Неучтенные факторы

- Социально-экономический статус
- Качество медицинской помощи
- Экологические факторы (загрязнение воздуха)
- Подробные психосоциальные стрессоры
- Гормональные изменения (менопауза, андропауза)
- Микробиом кишечника

## 10. Практическое применение модели

### 10.1 Для исследовательских целей

Модель позволяет:

- Изучить влияние отдельных факторов образа жизни
- Тестировать гипотезы о механизмах заболеваний
- Разрабатывать персонализированные рекомендации
- Создавать образовательные сценарии "что если"

### 10.2 Примеры расчетов

**Пример 1: Расчет изменения САД (Реалистичный сценарий)**

Исходные данные:

- Увеличение ИМТ: $\Delta BMI = 3$ кг/м²
- Увеличение потребления соли: $\Delta Na = 1200$ мг → $\Delta Na_{100mg} = 12$
- Потребление алкоголя: $Alc_{g/day} = 40$ г/день
- Дефицит сна: $AvgSleepHours = 5.5$ часа
- Увеличение стресса: $\Delta Stress = 4$ балла

Расчет:

1. Влияние алкоголя: $AlcEffect = 0.03 \times (40 - 10) = 0.9$
2. Влияние сна: $SleepEffect = (6 - 5.5) \times 0.5 = 0.25$
3. Изменение САД:

   $\Delta SBP = 0.7 \times 3 + 0.25 \times 12 + 0.03 \times 30 + 0.5 \times 4 + 0.5 \times 0.25$

   $\Delta SBP = 2.1 + 3.0 + 0.9 + 2.0 + 0.25 = 8.25$ мм рт.ст.
   *(Значение находится в реалистичном диапазоне годового изменения)*

**Пример 2: Расчет non-HDL cholesterol**

Исходные данные:

- Исходный TC = 200 мг/дл, исходный HDL = 50 мг/дл
- $\Delta TC = -4.49$ мг/дл
- $\Delta HDL = +2.85$ мг/дл

Расчет:

1. Новый TC = $200 - 4.49 = 195.51$ мг/дл
2. Новый HDL = $50 + 2.85 = 52.85$ мг/дл
3. non-HDL (мг/дл) = $195.51 - 52.85 = 142.66$ мг/дл
4. non-HDL (ммоль/л) = $142.66 / 38.67 = 3.69$ ммоль/л
5. После clipping: non-HDL остаётся в допустимых диапазонах.

**Пример 3: Расчет индекса дисбаланса КБЖУ и вклада в HbA1c**

Фактическое потребление: Белки 20%, Жиры 40%, Углеводы 40%

Расчет индекса:
$\Delta P = |20 - 30| = 10$
$\Delta F = |40 - 30| = 10$
$\Delta C = |40 - 40| = 0$

$raw\_KBJU = \sqrt{10^2 + 10^2 + 0^2} = \sqrt{200} \approx 14.14$

Нормализованное значение: $KBJU\_Imbalance = \min(100, 14.14 \times 10) = 141.4 \rightarrow 100$ (ограничение)

Вклад в HbA1c: $100 \times 0.002 = 0.2\%$

*Примечание:* В коде используется нормализованная шкала 0-100, поэтому для умеренного дисбаланса (raw=14.14, norm=100
из-за ограничения) вклад составляет 0.2%.

**Пример 4: Расчет риска ССЗ по упрощенной модели SCORE2 (С калиброванным интерцептом -3.8)**

Данные пациента:

- Возраст = 50 лет
- САД = 140 мм рт.ст.
- non-HDL = 4.2 ммоль/л
- Курение = 10 пачко-лет
- Диабет = нет (0)
- Генетический риск ССЗ = 1.2

Расчет:
$CVD_{\text{logit}} = -3.8 + 0.05 \times (50-40) + 0.03 \times (140-120) + 0.3 \times (4.2-3.5) + 0.08 \times 10 + 0.9 \times 0 + 0.4 \times 1.2$

$CVD_{\text{logit}} = -3.8 + 0.5 + 0.6 + 0.21 + 0.8 + 0 + 0.48 = -1.21$

$Risk_{CVD} = \frac{1}{1 + e^{-(-1.21)}} = \frac{1}{1 + e^{1.21}} \approx 0.230$ **(23.0%)**

*(Реалистичный риск для пациента средних лет с факторами риска; без курения риск был бы ~15%.)*

## 11. Заключение

Представленная модель представляет собой компромисс между научной точностью и вычислительной эффективностью. Она
основана на современных эпидемиологических данных и клинических рекомендациях, с калибровкой коэффициентов для
обеспечения баланса признаков в ML-моделях.

**Ключевые преимущества:**

- Механистическая основа: Имитация биологических процессов
- Использование валидированных клинических шкал: SCORE2, FINDRISC, INTERSALT
- Учет временной динамики: 20-летнее наблюдение с ежегодными измерениями
- Возможность моделирования интервенций: Сценарии "что если"
- Интеграция генетических факторов: Упрощенная модель полигенных рисков
- Разделение типов активности: кардио и силовые тренировки
- Учет мышечной массы: Влияние на метаболизм и риски
- Модель сна: Влияние дефицита сна на биомаркеры
- Индекс дисбаланса КБЖУ: Количественная оценка качества питания (0-100 шкала)
- **Шум измерений:** Добавлена биологическая вариативность для реалистичности ML-обучения
- **Калибровка рисков:** Скорректированы коэффициенты SCORE2, FINDRISC и натрия для адекватности прогнозов
- **Физиологическая корректность:** Исправлено распределение мышечной массы и двойного учета возраста
- **Упрощенная модель состава тела:** Явно документировано предположение о постоянной массе костей/органов

**Основное ограничение:**
Синтетическая природа данных требует осторожности при экстраполяции выводов на реальные клинические ситуации. Модель не
заменяет реальные клинические испытания.

**Дисклеймер**: Все формулы и коэффициенты основаны на опубликованных научных исследованиях с упрощениями и калибровкой,
специально сделанными для генерации синтетических данных для обучения ML-моделей. Модель предназначена для
исследовательских и образовательных задач ML и не заменяет клиническую оценку.

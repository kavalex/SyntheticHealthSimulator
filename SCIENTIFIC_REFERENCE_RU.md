# Научное описание модели генерации синтетических медицинских данных

## Введение

Представленная модель генерации синтетических медицинских данных основана на механистическом подходе, объединяющем
физиологические модели метаболизма, эмпирические эпидемиологические данные и клинические шкалы оценки риска. Модель
предназначена для имитации влияния образа жизни на здоровье человека в течение 20-летнего периода. Все коэффициенты
получены из мета-анализов, систематических обзоров и откалиброваны для обеспечения реалистичных годовых изменений.

**Проект:** `SyntheticHealthSimulator`

**ВАЖНО:** Данная модель содержит научно обоснованные упрощения и теоретические допущения, сделанные специально для
создания образовательного синтетического набора данных для обучения моделей машинного обучения. Она **НЕ** предназначена
для клинического применения или замены профессионального медицинского обследования.

## 1. Физиологические основы модели

### 1.1 Расчет базовой скорости метаболизма (Basal Metabolic Rate, BMR)

В модели используется уравнение Миффлина–Сен-Жеора:

**Формула Миффлина–Сен-Жеора:**

- **Мужчины**: $\mathrm{BMR}_{\mathrm{baseline}} = 10 \times \text{вес (кг)} + 6.25 \times \text{рост (см)} - 5 \times \text{возраст (годы)} + 5$
- **Женщины**: $\mathrm{BMR}_{\mathrm{baseline}} = 10 \times \text{вес (кг)} + 6.25 \times \text{рост (см)} - 5 \times \text{возраст (годы)} - 161$

**Коррекция на мышечную массу:**

$$\mathrm{BMR}_{\mathrm{adjusted}} = \mathrm{BMR}_{\mathrm{baseline}} + 8.5 \times (\mathrm{MuscleMass}_{\mathrm{кг}} - \mathrm{TypicalMuscleMass}_{\mathrm{кг}})$$

где $\mathrm{TypicalMuscleMass} = 0.3 \times \text{масса тела}$ (типичная мышечная масса составляет ~30% от массы тела),
а $8.5\,\mathrm{ккал/кг/день}$ представляет разницу в энерготратах между мышечной и жировой тканью в покое.

**Примечание:** Значение $0.3 \times$ массы тела используется как теоретический референс для расчета отклонения;
фактическая мышечная масса в данных может быть значительно ниже (см. Раздел 1.4). Это упрощение принято для
образовательных целей и не влияет на относительные сравнения.

**Примечание:** Уравнение Миффлина–Сен-Жеора уже включает возрастную поправку. Дополнительное снижение метаболизма с
возрастом моделируется через изменения мышечной массы (Раздел 1.4).

### 1.2 Расчет общих суточных энерготрат (Total Daily Energy Expenditure, TDEE)

Для учета физической активности используется система MET-минут:

$$\mathrm{TDEE} = \mathrm{BMR}_{\mathrm{adjusted}} \times \text{коэффициент активности}$$

**Коэффициенты активности по калиброванной классификации:**

| MET-минут/неделя | Коэффициент | Описание                              |
|------------------|-------------|---------------------------------------|
| $< 450$          | 1.2         | Сидячий/легкая активность (1-2 дня)   |
| $450$–$900$      | 1.375       | Умеренная активность (3-5 дней)       |
| $900$–$1500$     | 1.55        | Высокая активность (6-7 дней)         |
| $> 1500$         | 1.725       | Очень высокая активность (спортсмены) |

**Калибровка:** Пороги скорректированы от исходной классификации ВОЗ по дням недели к MET-минутам для обеспечения
реалистичного распределения популяции (~40% сидячий образ жизни, ~35% легкая активность, ~20% умеренная, ~5% высокая
активность). Коэффициент 1.9 для экстремальных нагрузок ($>3000$ MET-мин/нед) не используется — верхний порог ограничен
1.725 для предотвращения нефизиологичных значений TDEE $>4000$ ккал.

### 1.3 Модель изменения массы тела

#### 1.3.1 Энергетический баланс

Изменение веса рассчитывается с использованием механистической модели с временным шагом $\Delta t = 365$ дней (1 год):

$$\Delta W_{\mathrm{metabolic}} = \frac{(C - \mathrm{TDEE}) \times 365}{7700}$$

где:

- $\Delta W$ — изменение веса (кг) за год
- $C$ — потребление калорий (ккал/день)
- $\mathrm{TDEE}$ — энерготраты (ккал/день)
- $7700$ — калорийческий эквивалент 1 кг жировой ткани ($\mathrm{ккал/кг}$)

**Возврат к здоровому BMI (Mean Reversion):**

$$\mathrm{target\_BMI} = 23 + 0.1 \times \max(0, \mathrm{age} - 30)$$
$$\mathrm{reversion\_force} = (\mathrm{target\_weight} - \mathrm{current\_weight}) \times 0.04$$
$$\Delta W_{\mathrm{total}} = \Delta W_{\mathrm{metabolic}} + \mathrm{reversion\_force}$$

**Примечание:** Сила возврата ограничена $\pm 2.5$ кг/год для предотвращения нереалистичных изменений веса.

#### 1.3.2 Распределение изменения веса

$$\mathrm{ProteinFactor} = \min(0.4, 0.1 + 0.05 \times \mathrm{StrengthActivity} + 0.05 \times \mathrm{ProteinBalance})$$
$$\Delta W_{\mathrm{muscle}} = \Delta W \times \mathrm{ProteinFactor}$$
$$\Delta W_{\mathrm{fat}} = \Delta W \times (1 - \mathrm{ProteinFactor})$$

где $\mathrm{StrengthActivity}$ — количество силовых тренировок в неделю (0-5, рассчитывается
как $\text{strength\_met} / 150$), а $\mathrm{ProteinBalance}$: $1$ при потреблении $>1.6\,\mathrm{г/кг}$, $0$
при $0.8$–$1.6\,\mathrm{г/кг}$, $-1$ при $<0.8\,\mathrm{г/кг}$.

**Примечание:** Максимальная доля мышечной ткани в прибавке веса ограничена 40% для предотвращения физиологически
невозможных сценариев. При дефиците калорий мышечная масса может уменьшаться.

#### 1.3.3 Упрощенная модель состава тела

**Важное допущение:** В модели используется упрощенное представление состава тела:

$$\text{Вес} = \text{Жировая масса} + \text{Мышечная масса} + \text{Константа}$$

где Константа (костная масса, органы, вода) считается неизменной на протяжении всего периода моделирования. Это
упрощение необходимо для вычислительной эффективности и фокуса на динамике жировой и мышечной массы как ключевых
факторов метаболизма и риска. В реальности костная и органная масса также подвержены небольшим изменениям, но их вклад в
изменчивость риска минимален по сравнению с жировой и мышечной тканью.

**Формула обновления веса:**
$$\mathrm{Вес}_{t+1} = (\mathrm{Жировая\ масса}_t + \Delta W_{\mathrm{fat}}) + (\mathrm{Мышечная\ масса}_t + \Delta W_{\mathrm{muscle}}) + \text{Константа}$$

### 1.4 Модель мышечной массы

Мышечная масса в кг рассчитывается как:

$$\mathrm{МышечнаяМасса}_{t+1} = \mathrm{МышечнаяМасса}_t + \Delta W_{\mathrm{muscle}}$$

где $\Delta W_{\mathrm{muscle}}$ — изменение из Раздела 1.3.2.

**Индекс мышечной массы (MuscleMassFactor, 0-20)** используется для коррекции ИМТ:

$$\mathrm{MuscleMassFactor} = \mathrm{МышечнаяМасса}_{\mathrm{кг}} / 0.5 \quad \text{(1 единица индекса = 0.5 кг)}$$
$$\mathrm{ИМТ}_{\mathrm{корр}} = \mathrm{ИМТ} \times (1 - 0.01 \times \mathrm{MuscleMassFactor})$$

**Примечание:** В исходной когорте индекс мышечной массы генерируется как:

- Для мужчин: $\mathcal{N}(12, 3^2)$, ограничение $[0,20]$
- Для женщин: $\mathcal{N}(8, 2.5^2)$, ограничение $[0,20]$

Таким образом, средняя мышечная масса в кг составляет 6 кг для мужчин и 4 кг для женщин, что согласуется с упрощенной
моделью состава тела.

## 2. Модели биомаркеров

### 2.1 Систолическое артериальное давление (SBP)

#### 2.1.1 Формула изменения SBP (годовое изменение)

$$\Delta \mathrm{SBP} = \beta_{\mathrm{ИМТ}} \times \Delta \mathrm{ИМТ} + \beta_{\mathrm{Na}} \times \Delta \mathrm{Na}_{100\mathrm{мг}} + \beta_{\mathrm{Алк}} \times \mathrm{AlcEffect} + \beta_{\mathrm{Стресс}} \times \Delta \mathrm{Стресс} + \beta_{\mathrm{Сон}} \times \mathrm{SleepEffect} + \beta_{\mathrm{Возраст}} \times \mathrm{AgeEffect} + \mathrm{MeanReversion}$$

**Откалиброванные коэффициенты:**

- ИМТ: $+0.4$ мм рт.ст. на $1\,\mathrm{кг/м^2}$ (снижено с 0.7 для реалистичного среднего ~125 мм рт.ст.)
- Натрий: $+0.12$ мм рт.ст. на $100\,\mathrm{мг/день}$ (снижено с 0.25)
- Алкоголь: $+0.015$ мм рт.ст. на $1\,\mathrm{г/день}$ выше порога 10г (снижено с 0.03)
- Стресс: $+0.25$ мм рт.ст. на 1 единицу шкалы (снижено с 0.5)
- Сон: $+0.25$ мм рт.ст. на час дефицита сна (снижено с 0.5)
- Возраст: $+0.02$ мм рт.ст. на год выше 35

**Возврат к среднему (Mean Reversion):**

$$\mathrm{age\_norm\_SBP} = 115 + 0.3 \times \max(0, \mathrm{age} - 30)$$
$$\mathrm{reversion} = (\mathrm{age\_norm\_SBP} - \mathrm{current\_SBP}) \times 0.02$$

**Общее изменение ограничено $[-3.0, 4.0]$ мм рт.ст./год**

#### 2.1.2 Модель влияния алкоголя (калиброванная линейная)

Хронический эффект (среднее дневное потребление):
$$
\mathrm{AlcEffect} =
\begin{cases}
0, & \text{если } \mathrm{Алк}_{г/день} \leq 10 \\
0.015 \times (\mathrm{Алк}_{г/день} - 10), & \text{если } \mathrm{Алк}_{г/день} > 10
\end{cases}
$$

**Калибровка:** Коэффициент снижен для баланса признаков в ML-модели и предотвращения доминирования фактора алкоголя.

#### 2.1.3 Влияние сна (теоретическое допущение)

$$\mathrm{SleepEffect} = \max(0, 6 - \mathrm{СреднееЧасовСна}) \times 0.25$$

### 2.2 Холестерин ЛПВП (HDL)

#### 2.2.1 Формула изменения HDL (годовое изменение)

$$\Delta \mathrm{HDL} = \beta_{\mathrm{Cardio}} \times \mathrm{CardioActivity} + \beta_{\mathrm{Strength}} \times \mathrm{StrengthActivity} + \beta_{\mathrm{Fiber}} \times \Delta \mathrm{Fiber} + \beta_{\mathrm{Muscle}} \times \mathrm{MuscleMass} + \beta_{\mathrm{SFA}} \times \Delta \mathrm{SFA}_{\mathrm{ppt}} + \beta_{\mathrm{BMI}} \times \mathrm{BMIPenalty} + \beta_{\mathrm{Age}} \times \mathrm{AgePenalty} + \mathrm{SexFactor}$$

**Откалиброванные коэффициенты:**

- Аэробная активность: $+0.25$ мг/дл на $1000$ MET-минут
- Силовые тренировки: $+0.35$ мг/дл на $450$ MET-минут (≈3 занятия)
- Клетчатка: $+0.25$ мг/дл на $1\,\mathrm{г}$ изменения
- Мышечная масса: $+0.05$ мг/дл на 1 единицу индекса
- Насыщенные жиры (SFA): $-0.12$ мг/дл на 1 процентный пункт
- Штраф за BMI: $-0.15$ мг/дл на единицу выше 25
- Возрастной штраф: $-0.04$ мг/дл на год выше 35
- Половой фактор: $+0.3$ мг/дл для женщин

**Общее изменение ограничено $[-1.5, 0.8]$ мг/дл/год**

**Примечание:** За 20 лет это даёт реалистичное снижение HDL на -10 to -20 мг/дл суммарно.

**Примечание:** SFA хранится и передается в модели как доля (например, $0.30$ для $30%$). Для расчета изменения

### 2.3 Гликированный гемоглобин (HbA1c)

#### 2.3.1 Формула изменения HbA1c (годовое изменение)

$$\Delta \mathrm{HbA1c} = \mathrm{BaseDrift} + \mathrm{RiskDrift} + \mathrm{MeanReversion} + \sum \beta_i \times \mathrm{Factor}_i$$

**Откалиброванные компоненты:**

- Базовый дрейф: $+0.015%$ в год (базовый уровень для здоровой популяции)
- Дрейф риска: $\mathrm{DiabetesRisk} \times 0.15$ (дополнительный дрейф для групп высокого риска)
- Возврат к среднему: $(\mathrm{age\_norm} - \mathrm{current\_HbA1c}) \times 0.02$
    - где $\mathrm{age\_norm} = 5.0 + 0.02 \times \max(0, \mathrm{age} - 30)$
- Простые углеводы: $+0.03%$ на $10\,\mathrm{г}$ изменения
- ИМТ: $+0.006%$ на $1\,\mathrm{кг/м^2}$ изменения
- Клетчатка: $-0.0004%$ на $1\,\mathrm{г}$ изменения
- Физическая активность: $-0.002%$ на $1000$ MET
- MBI: $+0.0004%$ на 1 единицу индекса
- Мышечная масса: $-0.001%$ на 1 единицу индекса
- Дефицит сна: $+0.002%$ при сне $<6$ часов

**Общее изменение ограничено $[-0.1, 0.15]\%$/год**

**Ограничение диапазона:** Значения HbA1c строго ограничены диапазоном $3.5$–$12%$. Значения $>12%$ соответствуют
тяжелому неконтролируемому диабету и маловероятны для 20-летней когорты без летального исхода.

#### 2.3.2 Индекс баланса макронутриентов (Macronutrient Balance Index, MBI)

Индекс баланса макронутриентов (MBI) количественно оценивает отклонение от референсного распределения макронутриентов (
Белки $30%$, Жиры $30%$, Углеводы $40%$). Это нормализованный показатель от 0 (идеальный баланс) до 100 (максимальный
дисбаланс).

**Расчет индекса:**

$$\mathrm{raw\_MBI} = \sqrt{(\Delta P)^2 + (\Delta F)^2 + (\Delta C)^2}$$

где:

- $\Delta P = |\text{Фактический\_Белок}\% - 30|$
- $\Delta F = |\text{Фактический\_Жир}\% - 30|$
- $\Delta C = |\text{Фактический\_Углевод}\% - 40|$

**Нормализация:**
$$\mathrm{MBI} = \min(100, \mathrm{raw\_MBI} \times 10)$$

Умножение на 10 с ограничением 100 является техническим масштабированием для совместимости с логистическими моделями
риска и не представляет клинически валидированную шкалу.

### 2.4 Общий холестерин и не-ЛПВП (non-HDL)

**Формула изменения общего холестерина (годовое изменение):**

$$\Delta \mathrm{TC} = +1.5 \times \Delta \mathrm{SFA}_{\mathrm{ppt}} - 0.5 \times \Delta \mathrm{Fiber}_{1\mathrm{г}} - 0.3 \times \left(\frac{\mathrm{Activity}_{\mathrm{total}}}{1000}\right)$$

где $\Delta \mathrm{SFA}_{\mathrm{ppt}}$ — изменение в процентных пунктах (например, с $0.30$ до $0.31 = +1$ п.п.). SFA
передается в модель как доля (0.0–1.0), поэтому формула в коде: $1.5 \times (\Delta \mathrm{SFA} \times 100)$.

**Латентный липидный фактор:** Латентный липидный фактор генерируется на базовом уровне ($\mathcal{N}(0, 1)$, усечено
до $[-2, 2]$) и применяется к HDL и TC во все годы:

- $\mathrm{HDL}_{\mathrm{adjusted}} = \mathrm{HDL}_{\mathrm{base}} + 5.0 \times \mathrm{lipid\_factor}$
- $\mathrm{TC}_{\mathrm{adjusted}} = \mathrm{TC}_{\mathrm{base}} - 5.0 \times \mathrm{lipid\_factor}$

**Примечание:** Генетический риск влияет только на исходный уровень TC (при инициализации), а не на годовое изменение. В
коде $0.8 \times \mathrm{GeneticRisk}_{\mathrm{ССЗ}}$ добавляется к исходному уровню TC (в мг/дл), после чего
применяется ограничение до $150$–$350\,\mathrm{мг/дл}$. Это отражает биологическую предрасположенность, а не скорость
изменений под воздействием факторов риска.

**Расчет холестерина не-ЛПВП (non-HDL):**

1. Рассчитать TC в мг/дл по формуле выше
2. Рассчитать HDL в мг/дл по формуле из Раздела 2.2
3. non-HDL (мг/дл) = TC – HDL
4. non-HDL (ммоль/л) = (TC – HDL) / 38.67 (конверсия)

**Примечание:** При сильном измерительном шуме возможны отрицательные промежуточные значения; в коде
применяется $\mathrm{clip}(\text{non\_hdl\_mgdl}, 50, 232)$ перед конверсией в ммоль/л. Полученное значение в ммоль/л
затем дополнительно ограничивается диапазоном $[2.0, 6.0]$.

|-----------------------|-------------|----------------------------------------|-------------------------|-------------------------|
| Насыщенные жиры (SFA) | $+1.5$      | мг/дл на 1 процентный пункт            | Clarke et al., 1997     | —                       |
| Клетчатка             | $-0.5$      | мг/дл на $1\,\mathrm{г}$               | Brown et al., 1999      | —                       |
| Физическая активность | $-0.3$      | мг/дл на $1000$ MET-минут              | Kodama et al., 2007     | —                       |
| Генетика              | $+0.8$      | мг/дл на 1 единицу генетического риска | Hindy et al., 2020      | Только исходный уровень |
| Липидный фактор       | $\pm 5.0$   | мг/дл на 1 единицу                     | Теоретическое допущение | Применяется к HDL и TC  |

### 2.5 Измерительный шум и биологическая вариабельность

Для обеспечения реалистичности данных и предотвращения переобучения ML-моделей на идеальных формулах к конечным
значениям биомаркеров добавляется случайный шум.

**Обоснование:** Лабораторные анализы имеют погрешность измерения, а физиологические параметры колеблются ежедневно (
биологическая вариабельность).

**Формула шума:**

$$\mathrm{Значение}_{\mathrm{итог}} = \mathrm{Значение}_{\mathrm{расчетное}} + \mathcal{N}(0, \sigma^2)$$

**Параметры шума ($\sigma$):**

| Биомаркер        | $\sigma$                 |
|------------------|--------------------------|
| SBP              | $5\,\mathrm{мм\ рт.ст.}$ |
| HDL              | $5\,\mathrm{мг/дл}$      |
| Общий холестерин | $10\,\mathrm{мг/дл}$     |
| HbA1c            | $0.2%$                   |
| Вес              | $1.5\,\mathrm{кг}$       |

**Примечание:** Жесткое ограничение (hard clipping) применяется после добавления шума для обеспечения физиологических
ограничений (см. Раздел 8.3 для допустимых диапазонов). Это гарантирует, что все итоговые значения биомаркеров остаются
в клинически правдоподобных пределах.

## 3. Модели расчета медицинских рисков

### 3.1 Сердечно-сосудистые заболевания (модель на основе SCORE2)

**Упрощенная логистическая модель SCORE2:**

$$\mathrm{ССЗ}_{\mathrm{logit}} = \alpha + \beta_1 \times (\mathrm{Возраст}-40) + \beta_2 \times (\mathrm{SBP}-120) + \beta_3 \times (\text{non-HDL}-3.5) + \beta_4 \times \mathrm{Курение} + \beta_5 \times \mathrm{Диабет} + \beta_6 \times \mathrm{GeneticRisk}_{\mathrm{ССЗ}}$$

**Где коэффициенты:**

- $\alpha = \mathbf{-3.8}$ (Калибровочный свободный член для ~15-18% prevalence)
- $\beta_1 = 0.05$ (возраст за год)
- $\beta_2 = 0.03$ (SBP за мм рт.ст.)
- $\beta_3 = 0.3$ (non-HDL за ммоль/л)
- $\beta_4 = \mathbf{0.08}$ (курение за пачка-лет, калибровано)
- $\beta_5 = 0.9$ (наличие диабета)
- $\beta_6 = 0.4$ (генетический риск ССЗ)

**Обоснование калибровки:**

- Свободный член $-3.8$ обеспечивает ожидаемую заболеваемость ССЗ ~15-18% за 20 лет в популяции в возрасте $35\pm8$ лет.
- Коэффициент курения $0.08$ снижен по сравнению с эпидемиологическими данными ($0.6$), чтобы предотвратить
  доминирование фактора курения и обеспечить баланс признаков для ML. При $20$ пачка-лет вклад составляет $1.6$ (
  ОШ $\approx 5.0$), создавая заметный, но не подавляющий эффект.

**Расчет риска:**

$$\mathrm{Риск}_{\mathrm{ССЗ}} = \frac{1}{1 + e^{-\mathrm{ССЗ}_{\mathrm{logit}}}}$$

**Примечание:** Риск ограничен $[0, 0.5]$ для предотвращения экстремальных вероятностей. Значение non-HDL в ммоль/л
перед подстановкой в формулу дополнительно ограничивается диапазоном $[2.0, 6.0]$ в соответствии с Разделом 8.3.

### 3.2 Сахарный диабет 2 типа (модель на основе FINDRISC)

**Балльная система:**

| Фактор                                                | Баллы |
|-------------------------------------------------------|-------|
| Возраст $>45$ лет                                     | 2     |
| ИМТ $25$–$29.9$                                       | 1     |
| ИМТ $\geq 30$                                         | +2    |
| HbA1c $5.7$–$6.4%$                                    | 3     |
| HbA1c $\geq 6.5%$                                     | +5    |
| Курение                                               | 2     |
| Низкое потребление клетчатки ($<20\,\mathrm{г/день}$) | 1     |
| MBI $>20$                                             | 2     |
| Мышечная масса $<5$ единиц                            | 1     |
| Дефицит сна ($<6$ часов)                              | 2     |
| Генетический риск диабета $>1.5$                      | 3     |

**Формула расчета риска:**

$$\mathrm{Риск}_{\mathrm{Диабет}} = \frac{1}{1 + e^{-0.4 \times (\mathrm{ОбщийБалл} - \mathrm{порог})}}$$

где `порог` — калибровочный параметр, позволяющий регулировать общую распространенность диабета в синтетической
популяции. Значение по умолчанию (`порог = 16.5` в v1.1.3) обеспечивает распространенность диабета примерно $15%$ за
20-летний период, что соответствует целевым показателям для задач ML.

**Калибровка:** Коэффициент $0.4$ и значение порога эмпирически подобраны для получения реалистичного распределения
риска с учетом дополнительных факторов (MBI, мышечная масса, сон), расширяющих исходную шкалу FINDRISC.

**Примечание:** Риск ограничен $[0, 0.5]$ для предотвращения экстремальных вероятностей.

### 3.3 Дополнительные модели риска (расширение)

Модель включает дополнительные упрощенные логистические модели для обогащения набора данных задачами
мультиклассификации:

**Инсульт:**

$$\mathrm{logit} = 0.03 \times (\mathrm{Возраст}-50) + 0.015 \times (\mathrm{SBP}-120) + 0.3 \times (\mathrm{HbA1c}-5.5) + 0.15 \times \mathrm{Курение} + 0.03 \times (\mathrm{ИМТ}-25) + 0.4 \times \mathrm{Генетика} - 3.0$$

**НАЖБП (неалкогольная жировая болезнь печени):**

$$\mathrm{logit} = 0.02 \times (\mathrm{Возраст}-50) + 0.04 \times (\mathrm{ИМТ}-25) + 0.25 \times (\mathrm{HbA1c}-5.5) + 0.4 \times (\mathrm{SFA}-0.3) + 0.3 \times [\mathrm{Клетчатка}<25] + 0.4 \times \mathrm{Генетика} - 2.5$$

**Колоректальный рак:**

$$\mathrm{logit} = 0.06 \times (\mathrm{Возраст}-50) + 0.8 \times [\mathrm{Клетчатка}<20] + 0.6 \times [\mathrm{Алк}>140] + 0.7 \times \mathrm{Курение} + 0.5 \times [\mathrm{ИМТ}>30] + 0.9 \times \mathrm{Генетика} + \mathcal{N}(0, 0.3) - 1.2$$

**Цирроз:**

$$\mathrm{logit} = 0.015 \times (\mathrm{Возраст}-50) + 0.6 \times \ln(\mathrm{Алкоголь}+1) + 0.4 \times \mathrm{Курение} + 0.5 \times [\mathrm{ИМТ}>30] + 0.8 \times \mathrm{Генетика} + \mathcal{N}(0, 0.25) - 1.7$$

**Примечания:**

- Эти модели являются теоретическими допущениями для обогащения набора данных. Коэффициенты не калиброваны по
  эпидемиологическим данным, в отличие от SCORE2 и FINDRISC.
- Случайный шум ($\mathcal{N}(0, \sigma)$) добавлен к моделям рака и цирроза для увеличения вариативности.
- Отдельный генетический риск ожирения не реализован, так как ожирение моделируется через динамический ИМТ и факторы
  образа жизни.
- Значения рисков ограничены: Инсульт [0, 0.4], НАЖБП [0, 0.6], Рак [0, 0.3], Цирроз [0, 0.2]

### 3.4 Полигенные риски

#### 3.4.1 Генерация SNP-профиля (упрощенная)

Для каждого пациента набор SNP моделируется неявно путем извлечения логнормального фактора генетического риска. Этот
подход заменяет явное моделирование SNP для баланса между сложностью и вариативностью.

#### 3.4.2 Расчет полигенного риска (упрощенный)

Для каждого заболевания генетический риск извлекается из усеченного логнормального распределения:

$$\mathrm{GeneticRisk} \sim \mathrm{Логнормальное}(0, \sigma^2)$$, усечение до $[0.5, 2.0]$

#### 3.4.3 Распределения рисков:

| Заболевание        | Стандартное отклонение ($\sigma$) | Распределение (усеченное логнормальное) |
|--------------------|-----------------------------------|-----------------------------------------|
| ССЗ                | $0.4$                             | $\mathrm{Логнормальное}(0, 0.4^2)$      |
| Диабет             | $0.3$                             | $\mathrm{Логнормальное}(0, 0.3^2)$      |
| Колоректальный рак | $0.35$                            | $\mathrm{Логнормальное}(0, 0.35^2)$     |
| Инсульт            | $0.3$                             | $\mathrm{Логнормальное}(0, 0.3^2)$      |
| НАЖБП              | $0.25$                            | $\mathrm{Логнормальное}(0, 0.25^2)$     |
| Цирроз             | $0.4$                             | $\mathrm{Логнормальное}(0, 0.4^2)$      |

**Примечание:** Приведенные выше значения непосредственно используются в коде (метод `_generate_genetic_risk`). Явное
моделирование SNP не выполняется; логнормальный подход обеспечивает достаточную вариативность для задач ML.

### 3.5 Композитный показатель здоровья

Композитный показатель здоровья (шкала 0-100) рассчитывается из индивидуальных рисков заболеваний:

$$\mathrm{HealthScore} = 100 - \mathrm{ССЗ}_{\mathrm{risk}} \times 25 - \mathrm{Диабет}_{\mathrm{risk}} \times 20 - \mathrm{Инсульт}_{\mathrm{risk}} \times 20 - \mathrm{НАЖБП}_{\mathrm{risk}} \times 15 - \mathrm{Рак}_{\mathrm{risk}} \times 10 - \mathrm{Цирроз}_{\mathrm{risk}} \times 10$$

Показатель ограничен $[0, 100]$ и используется для:

- Оценки возраста события: $\mathrm{estimated\_event\_age} = 60 - 20 \times (1 - \mathrm{HealthScore}/100)$
- Определения основной причины смерти на основе порогов риска

## 4. Временные паттерны образа жизни

### 4.1 Автокоррелированные временные ряды (процесс Орнштейна–Уленбека)

Для генерации реалистичных траекторий используется возвращающаяся к среднему модель случайного блуждания:

$$X_{t+1} = X_t + \theta \times (\mu - X_t) + \sigma \times \epsilon_t$$

где:

- $\theta$ — скорость возврата к среднему
- $\mu$ — долгосрочное среднее (базовый уровень)
- $\sigma$ — волатильность
- $\epsilon_t \sim \mathcal{N}(0,1)$

**Конкретные параметры процесса ОУ, используемые в коде:**

| Фактор   | $\theta$ | $\sigma$ | Мин | Макс |
|----------|----------|----------|-----|------|
| Алкоголь | $0.8$    | $20.0$   | 0   | 500  |
| Аэробика | $0.2$    | $150.0$  | 0   | 3000 |
| Силовые  | $0.25$   | $120.0$  | 0   | 1500 |
| Курение  | $0.3$    | $0.3$    | 0   | 40   |
| Стресс   | $0.2$    | $1.0$    | 0   | 10   |
| Сон      | $0.2$    | $0.8$    | 4   | 10   |

**Примечание:** Theta алкоголя выше (0.8) чем у других факторов, что отражает более сильную устойчивость привычки.

### 4.2 Сезонные и случайные события

**Примечание:** Сезонные вариации не реализованы в коде (преднамеренное упрощение). При годовой дискретизации модели
сезонные колебания усредняются и не оказывают существенного влияния на долгосрочные траектории. Процессы ОУ (Раздел 4.1)
обеспечивают достаточную вариативность без явного моделирования сезонности.

**Случайные стрессовые события:**

- Вероятность стрессового события в любом данном году составляет $0.4$ (в среднем один раз в $2.5$ года)
- События (развод, потеря работы, болезнь) влияют на:
    - Уровень стресса: $+3$ балла на 1 год с затуханием ($+1.5$ на следующий год)
    - Качество сна: $-1$ час на 1 год
    - Потребление алкоголя: $+20%$ на 1 год

**Упрощение:** Продолжительность увеличена с $6$ месяцев (исходная спецификация) до $1$ года с затуханием для
соответствия годовой дискретизации модели.

## 7. Методология генерации данных

### 7.1 Иерархическая модель

Генетика → Демография → Образ жизни → Биомаркеры → Риски → Исходы

### 7.2 Формулы эволюции параметров

**Временной шаг:** Все изменения биомаркеров и веса рассчитываются за период $\Delta t = 365$ дней (1 год).

**Эволюция веса и состава тела:**

$$W_{t+1} = W_t + \Delta W_{\mathrm{жир}} + \Delta W_{\mathrm{мышцы}}$$

где $\Delta W_{\mathrm{жир}}$ и $\Delta W_{\mathrm{мышцы}}$ из Раздела 1.3.2.

**Эволюция мышечной массы:**

$$\mathrm{МышечнаяМасса}_{t+1} = \mathrm{МышечнаяМасса}_t + \Delta W_{\mathrm{мышцы}}$$

### 7.3 Генерация начальных условий

Для каждого синтетического пациента:

1. Генерация демографических данных (возраст, пол)
2. Генерация генетического профиля (логнормальные факторы риска для шести заболеваний)
3. Расчет исходных биомаркеров на основе демографии и генетики
4. Назначение исходного образа жизни (случайным образом с учетом корреляций)

**Демографические параметры:**

- Возраст: $\mathcal{N}(35, 8^2)$, усечено до $[20, 50]$
- Пол: распределение 50/50
- Рост (мужчины): $\mathcal{N}(178, 7^2)$ см
- Рост (женщины): $\mathcal{N}(163, 6^2)$ см

### 7.4 Требования к реализации в коде (Python)

**Баланс классов:** Код автоматически проверяет распределение целевой переменной (`has_disease`). Если дисбаланс
превышает 80/20, выдается предупреждение. Пороги риска можно регулировать через параметры генератора (например,
`diabetes_threshold`) для достижения желаемой распространенности.

**Пропущенные данные (MAR):** Для имитации реальных медицинских данных пропущенные значения (NaN) генерируются с
использованием механизма «пропущено случайно» (Missing At Random). Вероятность пропуска зависит от возраста и риска ССЗ:

$$p_{\mathrm{missing}} = 0.05 + 0.08 \times (\mathrm{age} < 45) - 0.05 \times (\mathrm{cvd\_risk\_10year} > 0.2)$$

ограничено до $[0.02, 0.15]$.

**Примечание:** Порог возраста — 45 лет (обновлено с 30 в более ранних версиях).

Пропуски применяются к выбранным столбцам биомаркеров во время агрегации (функция `_generate_mar_missing_values`).

**Этическая безопасность:** Все файлы и функции содержат предупреждение:
`# WARNING: SyntheticHealthSimulator - NOT FOR CLINICAL USE`.

**Явное указание калибровочных упрощений:** Ключевые коэффициенты аннотированы в коде комментариями, объясняющими
намеренное снижение для баланса признаков (например, курение, алкоголь, натрий, MBI).

**Жесткое ограничение (hard clipping):** Все биомаркеры ограничены диапазонами, перечисленными в Разделе 8.3.
Ограничение применяется сразу после расчета истинных значений и снова после добавления измерительного шума. Это
гарантирует, что все выходные значения физиологически правдоподобны.

**Фильтрация по границам:** Пациенты с биомаркерами вне физиологических диапазонов в финальном году отфильтровываются.
Типичный уровень сохранения ~40-50% (коэффициент оверсемплинга ~2.41).

## 8. Ограничения модели

### 8.1 Упрощения

- **Линейность:** $\Delta \mathrm{Биомаркер} = \sum \beta_i \times \mathrm{Фактор}_i$ (упрощение для образовательной
  модели)
- **Аддитивность:** Эффекты факторов суммируются (упрощение для обучения ML)
- **Независимость:** Корреляции между факторами упрощены
- **Типы активности:** Разделение на аэробные и силовые тренировки является теоретическим допущением
- **non-HDL:** SCORE2 использует измеренный non-HDL, в модели это упрощено как TC – HDL
- **Общий холестерин:** Упрощенный расчет без разделения на ЛПНП и ЛПОНП
- **Модель SCORE2:** Упрощенная логистическая модель аппроксимирует исходный полином
- **Состав тела:** Упрощенная модель с постоянной массой костей/органов (см. Раздел 1.3.3)
- **Липидный фактор:** Латентный фактор применяется равномерно по всем годам (упрощение генетических/средовых эффектов)

### 8.2 Теоретические допущения (следует учитывать при интерпретации)

- Влияние мышечной массы на BMR, HDL и HbA1c — теоретические коэффициенты, не подтвержденные крупными мета-анализами
- Влияние сна на SBP и HbA1c — основано на ассоциативных исследованиях, механистическая связь упрощена
- Индекс баланса макронутриентов (MBI) — оригинальная метрика модели, не валидированная в клинических исследованиях;
  нормализация ($\times 10$, ограничение 100) является техническим масштабированием для совместимости с ML
- Влияние силовых тренировок на HDL — коэффициент $+0.35\,\mathrm{мг/дл}$ является теоретическим допущением,
  откалиброванным через 450 MET-минут
- Модель полигенного риска — упрощенное представление реальной полигенной архитектуры заболеваний (логнормальное
  распределение вместо явного моделирования SNP)
- Отсутствие сезонности — годовая дискретизация модели делает сезонные колебания незначительными; вариативность
  обеспечивается процессами ОУ
- Дополнительные модели риска (инсульт, НАЖБП, рак, цирроз) — теоретические допущения для обучения ML, не калиброванные
  по эпидемиологическим данным
- Упрощенная модель состава тела — масса костей и органов предполагается постоянной (см. Раздел 1.3.3)
- Возврат к среднему в весе и SBP — теоретическое допущение для предотвращения нарушений границ

- Социально-экономический статус

- Факторы окружающей среды (загрязнение воздуха)
- Детальные психосоциальные стрессоры
- Гормональные изменения (менопауза, андропауза)
- Микробиом кишечника
- Влияние медикаментов
- Острые заболевания (кроме как стрессовые события)

## 10. Практическое применение модели

### 10.1 Для исследовательских целей

Модель позволяет:

- Изучать влияние отдельных факторов образа жизни
- Проверять гипотезы о механизмах заболеваний
- Разрабатывать персонализированные рекомендации
- Создавать образовательные сценарии «что, если»
- Обучение ML-моделей на реалистичных лонгитюдных данных
- Тестирование механизмов пропущенных данных (MAR)
- Практика обработки дисбаланса классов

### 10.2 Примеры расчетов

#### Пример 1: Расчет изменения SBP

**Исходные данные:**

- Увеличение ИМТ: $\Delta \mathrm{ИМТ} = 3\,\mathrm{кг/м^2}$
- Увеличение потребления соли: $\Delta \mathrm{Na} = 1200\,\mathrm{мг}$ → $\Delta \mathrm{Na}_{100\mathrm{мг}} = 12$
- Потребление алкоголя: $\mathrm{Алк}_{г/день} = 40\,\mathrm{г/день}$
- Дефицит сна: $\mathrm{СреднееЧасовСна} = 5.5$ часов
- Увеличение стресса: $\Delta \mathrm{Стресс} = 4$ балла
- Возраст: 50 лет
- Текущий SBP: 130 мм рт.ст.

**Расчет:**

- Эффект алкоголя: $\mathrm{AlcEffect} = 0.015 \times (40 - 10) = 0.45$
- Эффект сна: $\mathrm{SleepEffect} = (6 - 5.5) \times 0.25 = 0.125$
- Эффект ИМТ: $0.4 \times 3 = 1.2$
- Эффект натрия: $0.12 \times 12 = 1.44$
- Эффект стресса: $0.25 \times 4 = 1.0$
- Эффект возраста: $0.02 \times (50 - 35) = 0.3$
- Возврат к среднему: $(115 + 0.3 \times 20 - 130) \times 0.02 = -0.18$

**Изменение SBP:**

$$\Delta \mathrm{SBP} = 1.2 + 1.44 + 0.45 + 1.0 + 0.125 + 0.3 - 0.18 = 4.335\,\mathrm{мм\ рт.ст.}$$

**После ограничения:** $\Delta \mathrm{SBP} = 4.0\,\mathrm{мм\ рт.ст.}$ (максимально допустимое)

#### Пример 2: Расчет холестерина non-HDL

**Исходные данные:**

- Базовый TC = $200\,\mathrm{мг/дл}$, базовый HDL = $50\,\mathrm{мг/дл}$
- Липидный фактор = $0.5$
- $\Delta \mathrm{TC} = -4.49\,\mathrm{мг/дл}$
- $\Delta \mathrm{HDL} = +2.85\,\mathrm{мг/дл}$

**Расчет:**

- Скорректированный базовый TC = $200 - 5.0 \times 0.5 = 197.5\,\mathrm{мг/дл}$
- Скорректированный базовый HDL = $50 + 5.0 \times 0.5 = 52.5\,\mathrm{мг/дл}$
- Новый TC = $197.5 - 4.49 = 193.01\,\mathrm{мг/дл}$
- Новый HDL = $52.5 + 2.85 = 55.35\,\mathrm{мг/дл}$
- non-HDL (мг/дл) = $193.01 - 55.35 = 137.66\,\mathrm{мг/дл}$
- non-HDL (ммоль/л) = $137.66 / 38.67 = 3.56\,\mathrm{ммоль/л}$

**После ограничения:** non-HDL остается в допустимых пределах $[2.0, 6.0]\,\mathrm{ммоль/л}$

#### Пример 3: Расчет риска ССЗ с использованием упрощенной модели SCORE2

**Данные пациента:**

- Возраст = 50 лет
- SBP = $140\,\mathrm{мм\ рт.ст.}$
- non-HDL = $4.2\,\mathrm{ммоль/л}$
- Курение = 10 пачка-лет
- Диабет = 0
- Генетический риск ССЗ = 1.2

**Расчет:**

$$\mathrm{ССЗ}_{\mathrm{logit}} = -3.8 + 0.05 \times (50-40) + 0.03 \times (140-120) + 0.3 \times (4.2-3.5) + 0.08 \times 10 + 0.9 \times 0 + 0.4 \times 1.2$$
$$\mathrm{ССЗ}_{\mathrm{logit}} = -3.8 + 0.5 + 0.6 + 0.21 + 0.8 + 0 + 0.48 = -1.21$$
$$\mathrm{Риск}_{\mathrm{ССЗ}} = \frac{1}{1 + e^{-(-1.21)}} = \frac{1}{1 + e^{1.21}} \approx 0.230 \ (23.0\%)$$

**После ограничения:** Риск = 23.0% (в пределах $[0, 0.5]$)

(Реалистичный риск для пациента среднего возраста с факторами риска; без курения риск был бы ~15%.)

## 12. Заключение

Представленная модель представляет собой компромисс между научной точностью и вычислительной эффективностью. Она
основана на современных эпидемиологических данных и клинических рекомендациях с калибровкой коэффициентов для
обеспечения баланса признаков в моделях ML.

**Ключевые преимущества:**

- **Механистическая основа:** Моделирование биологических процессов
- **Использование валидированных клинических шкал:** SCORE2, FINDRISC, INTERSALT
- **Учет временной динамики:** 20-летнее наблюдение с ежегодными измерениями
- **Возможность моделирования вмешательств:** Сценарии «что, если»
- **Интеграция генетических факторов:** Упрощенная модель полигенного риска
- **Разделение типов активности:** аэробные и силовые тренировки
- **Учет мышечной массы:** Влияние на метаболизм и риски
- **Модель сна:** Влияние дефицита сна на биомаркеры
- **Индекс баланса макронутриентов (MBI):** Количественная оценка качества диеты (шкала 0-100)
- **Измерительный шум:** Добавленная биологическая вариабельность для реалистичного обучения ML
- **Калибровка рисков:** Скорректированные коэффициенты SCORE2, FINDRISC и натрия для адекватных предсказаний
- **Физиологическая корректность:** Скорректированное распределение мышечной массы и двойной учет возраста
- **Механизмы возврата к среднему:** Предотвращают нарушения границ в весе и SBP
- **Латентный липидный фактор:** Создает реалистичную структуру корреляции HDL-TC
- **Комплексная фильтрация по границам:** Обеспечивает итоговое качество данных
- **Пропущенные значения MAR:** Реалистичные паттерны пропусков для обучения ML

**Основное ограничение:**
Синтетическая природа данных требует осторожности при экстраполяции выводов на реальные клинические ситуации. Модель не
заменяет реальные клинические испытания.

**Отказ от ответственности:** Все формулы и коэффициенты основаны на опубликованных научных исследованиях с упрощениями
и калибровкой, специально выполненными для генерации синтетических данных для обучения моделей ML. Модель предназначена
для исследований в области ML и образовательных задач и не заменяет клиническую оценку.
